## Explore Shanti Nagar Now!
A full-stack website which acts as a tourist guide for people who would like to explore Shanti Nagar and its surrondings.

## Tech Stack

- Frontend: React (Vite), Tailwind CSS, JavaScript.
- Backend: Node.js, Express.js, Mongoose.js.
- Database: MongoDB.
- Database as a Service (DBaaS): MongoDB Atlas (Cloud) 
- Deployments: Vercel.

- [Website link](https://explore-shanthi-nagar.vercel.app)
